# Notes

Training notes, TryHackMe writeups, and lab references. This directory is part of the `huginn` vault — open the repo root in Obsidian.

## Structure

```
notes/
├── templates/          # Obsidian templates (Engagement Kickoff, Recon, Finding, THM)
├── training/           # TryHackMe rooms, HTB, lab exercises
│   ├── thm/
│   └── htb/
└── research/           # Vulnerability research, tool deep-dives
```

## Private Engagements

Client work lives in a separate local-only vault at `~/vaults/ops-private/` — never pushed to GitHub.
